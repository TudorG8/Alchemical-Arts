# Alchemical Arts

* This project is showcasing a fluid simulation (SPH) created in Unity DOTS, using jobs to attain good performance even at 10000 particles.

* The new low level physics module from Unity 6.3 was used to achieve good physics performance in 2D to allow the liquid to collide with any shapes.

# Controls

* Left-Click - gather up particles
* Right-click - push away particles
* Scrollwheel - change area of input
